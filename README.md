# 🎮 Tic Tac Toe Game

A responsive and interactive Tic Tac Toe game built using **HTML, CSS, and JavaScript**. The project provides a clean user interface with smooth gameplay, making it a great beginner-friendly JavaScript project.

---

## 🚀 Live Demo

🔗 https://anubhav-1921.github.io/TIC-TAC-TOE-GAME-/

---

## ✨ Features

- 🎮 Two-player gameplay
- ✅ Automatic win detection
- 🤝 Draw detection
- 🔄 Restart game functionality
- 📱 Responsive design
- 🎨 Clean and user-friendly interface

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript

## 📖 How to Run

1. Clone the repository

```bash
git clone https://github.com/anubhav-1921/TIC-TAC-TOE-GAME-.git
```

2. Open the project folder.

3. Run `index.html` in your browser.
